Docker esempio01 nginx

Benvenuto! Questa pagina è servita da un container Nginx gestito con Docker e Terraform.

Proprietà dei componenti

  • Docker: Esegue e gestisce container in modo isolato e portabile.
  • Terraform: Definisce e orchestra infrastrutture come codice, anche per ambienti containerizzati.
  • Nginx: Web server leggero e performante, ideale per servire contenuti statici e come reverse proxy.
